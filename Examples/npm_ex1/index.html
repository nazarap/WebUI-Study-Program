Install lodash. Config webpack and babel

Code example

1. Install Node and npm on your PC

How to install Node and npm
About Node
About npm

2. Initialize npm and create package.json

$ npm init

Install a dependency by name. For example

$ npm install (package_name) -S

3. Install lodash like dependency

$ npm install lodash -S

You can find lodash package in your project by this path ./node_modules/lodash

4. Install webpack, babel like dev-dependency for build our project

$ npm install babel-loader @babel/core @babel/preset-env webpack -D

Babel, webpack example

5. Create webpack.config.js file in project root directory

6. Put this code into file webpack.config.js

7. Add script into ypur package.json file

scripts: { "webpack": "webpack --mode development" }

Example: package.json

8. Import lodash package in app.js

import lodash from 'lodash';

9. And try to use any function from lodash library

const square = n => n * n;
const print = value => console.log(value);

print( lodash.map([4, 8], square) );
// => [16, 64]

print( lodash.map({ 'a': 4, 'b': 8 }, square) );
// => [16, 64] (iteration order is not guaranteed)

const users = [
{ 'user': 'barney' },
{ 'user': 'fred' }
];

// The `_.property` iteratee shorthand.
print( lodash.map(users, 'user') );
// => ['barney', 'fred']

Example: app.js

10. Run command in Node command ot terminal

$ npm run webpack

In ./build folder we have new file app.bundle.js

11. Now we have to import this file from index.html

<script src="./build/app.bundle.js" type="application/javascript"></script>
