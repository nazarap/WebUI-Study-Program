React application example

React app

1. Install React and React dom

$ npm run react react-dom -S

2. Install Babel present for react

$ npm i @babel/preset-react -D

3. Add react present to presents array in webpack.config.js

presets: ['@babel/preset-react', '@babel/preset-env']

4. Import react and react dom render method in app.js file

import React from 'react';
import { render } from 'react-dom';

5. Add code for render some html

render(
<h1>Hello, world!</h1>,
document.getElementById('root')
);

6. Add element with id root into index.html

<main id="root"></main>

7. Build our application

$ npm run webpack

8. Now we have to import this file from index.html

<script src="./build/app.bundle.js" type="application/javascript"></script>
